# Praya Studio — Website

Company profile site for **Praya Studio** (production experts collective, Jogja).

Static site (HTML/CSS/JS). Bold-editorial aesthetic mirroring the studio's company profile — black / cream / olive, giant display type, premium scroll motion (smooth scroll, text reveals, marquee, animated stats).

# → http://localhost:5050
